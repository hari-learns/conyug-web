## Design Requirements

### Style Guidelines
- **Modern & Vibrant**: Use gradients, glassmorphism, smooth animations
- **Indian Cultural Touch**: Incorporate subtle patterns, warm colors inspired by Indian aesthetics
- **Mobile-First**: Must be fully responsive (most Indian users are mobile-first)
- **Performance**: Fast loading, optimized images
- **Accessibility**: High contrast, readable fonts, proper alt texts

### Technical Stack
- HTML5 + Tailwind CSS (via CDN for simplicity)
- Vanilla JavaScript for interactions
- Smooth scroll animations (AOS library or CSS animations)
- Responsive images with lazy loading

### Key Visual Elements
- Animated hero section with subtle movement
- Hover effects on cards and buttons
- Smooth transitions between sections
- Illustrated icons for features (use lucide icons or custom)
- Testimonial/use case cards with real scenarios
- Map visualization showing "local" concept

### Typography
- Headings: Bold, modern sans-serif (Poppins or Inter)
- Body: Clean, readable (Outfit or System UI)
- Support for Devanagari and South Indian scripts

### Animation Preferences
- Fade-in-up on scroll
- Subtle parallax on hero
- Counter animations for stats
- Hover scale effects on cards
- Smooth color transitions
